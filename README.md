# CLI Mortgage Calculator

A simple and minimal command-line interface (CLI) based Mortgage Calculator program. This was developed by Hussain as a part of a university project.

### Overview

This tool allows you to perform common mortgage calculations directly from your terminal.

### Features: 

This calculator can determine the following:

**Monthly Payment**: The total amount to be paid each month.

**Interest Portion**: The part of your monthly payment that goes towards interest.

**Principal Portion**: The part of your monthly payment that goes towards paying down the loan principal.

**Remaining Balance**: The outstanding loan balance after a certain number of payments.
